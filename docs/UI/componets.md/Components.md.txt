This UI Component Library is designed to accompany the modern, scientific, and trustworthy Chemistry AI application. Developed in alignment with our 8pt Grid System and bilingual typography rules (Inter + Hind Siliguri), these components guarantee visual consistency, accessible tap targets, and high-quality technical presentation across both Light and Dark modes.
🛠️ Global Component Specifications
• Tap Targets: Any interactive component (buttons, switches, tab items) must maintain a minimum bounding box of $48\text{ dp} \times 48\text{ dp}$ for physical accessibility.

• Dividers: $1\text{ dp}$ width, colored #E2E8F0 (Light Mode) or #1E293B (Dark Mode).
