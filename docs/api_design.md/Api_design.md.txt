Global Design Choices
• Base URL: [https://api.rxnlearn.com/v1](https://api.rxnlearn.com/v1)

• Format: application/json for all requests and responses.

• Authentication: Stateless JWT Bearer Token passed in the Authorization header.

• Standard Error Wrapper: All 4xx and 5xx responses follow this structure:
JSON

{
  "error": {
    "code": "RESOURCE_NOT_FOUND",
    "message": "The requested reaction ID does not exist.",
    "details": []
  }
}

1. Authentication (/auth)
Register a User
• Endpoint: POST /auth/register

• Request Body:
JSON

{
  "name": "Jane Doe",
  "email": "jane.doe@example.com",
  "password": "StrongPassword123!"
}

• Response: 201 Created
JSON

{
  "message": "User registered successfully.",
  "user_id": "usr_9481a7b"
}

Log In User
• Endpoint: POST /auth/login

• Request Body:
JSON

{
  "email": "jane.doe@example.com",
  "password": "StrongPassword123!"
}

• Response: 200 OK
JSON

{
  "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "Bearer",
  "expires_in": 3600
}

4. Quizzes (/quizzes)
Get a Random/Targeted Quiz
• Endpoint: GET /quizzes

• Query Parameters: difficulty=Intermediate&limit=1

• Response: 200 OK
JSON

{
  "quiz_id": "qiz_8832",
  "title": "Organic Mechanisms Quiz",
  "questions": [
    {
      "question_id": "qn_01",
      "text": "Which solvent is essential for a Grignard reaction to prevent decomposition?",

        {"id": "b", "text": "Ethanol"},
        {"id": "c", "text": "Anhydrous Diethyl Ether"},
        {"id": "d", "text": "Acetone"}
      ]
    }
  ]

• Request Body:
JSON

    {
